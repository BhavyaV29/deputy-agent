# Media

The canonical walkthrough is the **live browser demo** (no install):
<https://deputy-web-demo.onrender.com>

A short screen recording / GIF is optional polish for README embeds. Prefer a real
capture of the live demo (plan → approval → audit) over fabricated screenshots.
If you add one, drop it here as `demo.gif` or `demo.mp4` and link it from the
README Demo section.

CI (pytest + ruff + mypy + web-demo parser tests) is the automated proof the
repo stays green — see `.github/workflows/ci.yml`.
